# 🌾 Trang Trại SEF

*Chào mừng mấy con bò của anh!* 🐄 

Welcome to the **Trang Trại SEF** (SEF Farm) project repository. This guide will help you set up your local development environment and get the web application running.

---

##🛠️ Installation & Setup Guide

Follow these steps to get the environment ready on your local machine.
1. Prerequisites

Before starting, ensure you have Node.js installed. This project requires it to manage packages and run the development server.

    Check Node.js version:
    Open your terminal and run:
    Bash

    node --version

    Download: If you don't have it, download the LTS version from nodejs.org.

2. Repository Setup

Clone this project to your local machine and navigate into the project folder:
Bash

git clone https://github.com/your-username/trang-trai-sef.git
cd trang-trai-sef

3. Install Dependencies

This project uses npm to manage libraries. Run the following command to download all necessary folders (like node_modules):
Bash

npm install

🚀 Running the Project
Start the Web Application

Once the installation is finished, you can launch the local development server:
Bash

npm run dev

    View the App: After running the command, the terminal will provide a local URL (usually http://localhost:5173 or http://localhost:3000).

    Stop the Server: Press Ctrl + C in your terminal to stop the application.

📁 Project Structure (Quick Look)

    src/: Contains the main source code for the web app.

    public/: Static assets like images and icons.

    package.json: Lists the project dependencies and scripts.
